JavaScript Callback

JavaScript functions are executed in the sequence they are called. Not in the sequence they are defined.

Sequence Control

To have a better control over when to execute a function, save the result then call another function to display the result.

JavaScript Callback

A callback is a function passed as an argument to another function

Using the callback function, you can call the exam result and let the exam result function run the callback after the calculation is finished

When to use Callback function

Callbacks are really good in asynchronous functions, where one function has to wait for another function- like waiting for a file to load

Asynchronous JavaScript

Functions that run parallel with other functions are called asynchronous a good example is JavaScript setTimeout()
